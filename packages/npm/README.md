# @calcom/cal-sans-ui

Cal Sans UI, Cal Sans Text, and Cal Sans Geo typefaces for [Cal.com](https://cal.com).

## Install

```bash
npm i @calcom/cal-sans-ui
```

**Next.js:** peer dependency Next.js 13.2+

For full installation and usage documentation (Next.js App Router, CSS imports, variable font axes, exports), see the [main repository README](https://github.com/calcom/sans-ui#readme).
